## Описание
**RCCarsScoreTable** - это программа, которая помогает считать результы гонок в игре Недетские гонки(Rc Cars). Программа нужна преимущественно тем, кто играет онлайн и считает общий результат гонок в конце игры по очкам.

Скачать программу можно в разделе **[Releases](https://github.com/MaxsonDev/rccars_tables_score_counter/releases/tag/1.0)** данного репозитория.

## Как пользоваться?
> [!WARNING]
> Программа работает не со всеми исполняемыми файлами <ins>Недетских гонок(Rc Cars)</ins>!
> 
> Работоспособность проверялась на версии игры из из магазина **IndieGala**. Совместимость 100%.

- Чтобы сохранить результат заезда, нажмите на клавишу **F5**. Лучше нажимать, когда финишировал хотя бы предпоследний игрок.
- Чтобы посчитать и отобразить результат всех заездов нажмите на кнопку **Посчитать результат**.
- Чтобы обновить таблицу и отобразить все сохраненные результаты нажмите на кнопку **Обновить**.
- Чтобы удалить лишнюю карту или случайно добавленную запись, выберите карту в таблице **Карта** и нажмите на кнопку **Удалить**.
